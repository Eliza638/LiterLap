# LiterLap

**Descripción del proyecto:**  
_LiterLap_ es un videojuego educativo desarrollado como recurso didáctico para reforzar el uso correcto de la acentuación en palabras agudas, graves y esdrújulas. Está dirigido a estudiantes del tercer grado de primaria, en un contexto rural de Nicaragua. Su propósito es promover el aprendizaje significativo a través del juego.

## 🌿 Ramas del repositorio:
- `main`: rama principal y estable
- `Interfaz`: contiene el archivo `pantalla-principal.html` y el archivo de estilos `estilos.css`
- `logica-juego`: contiene el archivo `logica-juego.js` con la base de la lógica del juego
- `Personajes`: destinada al diseño gráfico y personajes visuales del juego

---

## ⚙️ Tecnologías y herramientas utilizadas:
- **Lenguajes:** HTML, CSS, JavaScript
- **Gestión del código:** Git y GitHub (control de versiones)
- **Entorno de desarrollo:** GitHub Web

---

## 🎮 Objetivo técnico:
Desarrollar una aplicación educativa, interactiva y accesible, que apoye a los estudiantes a comprender la acentuación ortográfica, integrando las TIC al proceso de enseñanza-aprendizaje desde un enfoque lúdico y por competencias.

---

## 🧩 Estructura básica:
- `pantalla-principal.html`: página inicial del videojuego con mensaje de bienvenida.
- `estilos.css`: hoja de estilo con colores y fuentes aplicados al diseño.
- `logica-juego.js`: función base para clasificar palabras según su acentuación.
- `README.md`: documento técnico con los detalles del proyecto y sus avances.
